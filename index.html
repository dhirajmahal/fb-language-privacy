Privacy Policy – Language Test App

Last Updated: 27 November 2025

This application (“the App”) is a personal project created and operated by Dhiraj Mahal for testing and experimenting with Facebook’s Webhooks, AI content moderation, and automation features. The App is not intended for commercial use or for distribution to other users.

Information We Receive

The App may receive certain data from Facebook when events occur on a Facebook Page you manage, such as:

  • New comments
  • Edited comments
  • Hidden or deleted comments
  • Other page feed–related webhook events

This information is sent directly from Facebook via the Webhooks system.

How Data Is Used

All data received is used only for testing and automation purposes, specifically:

  • Automatically classifying comments using AI models
  • Hiding, deleting, or replying to comments based on moderation rules
  • Logging and debugging webhook events

No data is stored permanently and no data is shared with any third parties.

Data Storage

The App does not store any personally identifiable information in a database or external storage. Webhook events may be temporarily printed to the developer’s local console for debugging, but no long-term storage is used.

Data Sharing

No data processed by the App is sold, rented, or shared with third parties. All processing happens locally on the developer’s machine or in the developer’s own cloud resources solely for testing.

Security

All communication from Facebook is received over HTTPS via a secure tunnel (such as ngrok). Webhook requests are verified using a secret token to ensure that events originate from Facebook.

User Rights

Because the App does not store personal information, there is no user data to access, modify, or delete. If you have concerns about any content on the Facebook Page itself, you may contact the Page administrator directly using Facebook’s tools.

Children’s Privacy

The App is not directed toward children and does not knowingly collect information from children.
